# 项目简介

## 核心功能
- 基于YOLOv8的交通手势检测系统，可识别8类手势：变道、左转、左转等待、直行、靠边停车、右转、减速、停车
- 支持多种输入方式：单张图片、图片文件夹、视频文件、本地摄像头
- 提供UI界面，可实时显示目标位置、目标总数、置信度等信息
- 支持自定义检测参数：置信度阈值、IOU阈值
- 支持选择不同检测模型
- 支持批量检测并查看所有结果
- 支持检测结果保存功能

## 适用场景
- 交通手势识别研究
- 智能交通系统开发
- 计算机视觉教学演示
- 自动驾驶辅助系统原型开发

## 技术特点
- 技术栈：Python3.9 + OpenCV + PyQt5 + Ultralytics + PyTorch1.9
- 包含完整源码、数据集(4993张标注图片)、预训练模型及运行教程
- 提供图形化用户界面，操作便捷
- 已训练好模型，配置环境后可直接使用
